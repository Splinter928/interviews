10. Написать функцию, которая выведет сумму чисел последнего вложенного массива. Степень вложенности может изменяться
```python
arr = [1, [2, [3, [3, 5, 6]]]]

from typing import List
def rsum(arr: List[int]) -> int:
    if all(isinstance(k, int) for k in arr):
        return sum(arr)
    return rsum(arr[-1])
```
